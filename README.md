## 레스토랑 정보 예약 서비스 REST-API

### 무엇이 문제인가?

- 오늘 뭐 먹지?

**Why?**

- 주변에 어떤 가게가 있는지 잘 모른다.
- 매일 같은 곳에 가는 건 싫다
- 어떤 가게가 좋은지 잘 모르겠다.
- 기존에 방문한 사람들의 리뷰를 보고싶다
- 예전에 봐둔 가게가 있었는데 기억이 안난다.
- 가보고 싶은 곳을 기록하고 싶다.
- 가게에 자리가 있는지 모르겠다.
- 온라인으로 예약을 하고 싶다.

**사용자**

- 고객
  - 가게 목록 조회
  - 가게의 메뉴 조회
  - 가게의 평점 확인
  - 가게 목록 분류 조회
  - 평점 리뷰 남기기
  - 즐겨 찾기 추가
  - 예약 서비스
- 가게
  - 예약 요청 조회
  - 예약 요청 응답
- 관리자
  - 가게 정보 등록
